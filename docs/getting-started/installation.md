# Installation

Generative Computational Hallucinatory Art requires Python 3.11 or later.

## Install from PyPI

The easiest way to install peyote is from PyPI:

```bash
pip install peyote
```

## Install from Source

You can also install from source:

```bash
git clone https://github.com/crossjam/peyote.git
cd peyote
pip install -e .
```

## Development Installation

For development, we recommend using `uv`:

```bash
git clone https://github.com/crossjam/peyote.git
cd peyote
uv sync
```

This will install all dependencies including development tools.

## Verify Installation

After installation, verify that peyote is working:

```bash
peyote --version
```

You should see the version number displayed.
