## Overview

Generative Computational Hallucinatory Art is a Python package that provides a command-line interface for [brief description of what your package does].

## Quick Start

Install peyote using pip:

```bash
pip install peyote
```

Then run the CLI:

```bash
peyote --help
```

## Features

- Modern Python packaging with `uv` support
- CLI interface built with Typer
- Structured logging with Loguru
- Configuration management with Pydantic Settings
- Type checking with ty
- Code quality tools (ruff, pytest)
- Automated testing and CI/CD
